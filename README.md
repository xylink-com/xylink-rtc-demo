# XYLINK WEBRTC SDK
小鱼易连webrtc sdk demo项目。

此项目使用到的技术栈：react+typescript+hook

### 准备工作
1. 注意在`src/utils/config.ts`中配置第三方登录所需的
  * extId
  * clientId
  * clientSecret

### 运行

#### 安装依赖
```bash
$ yarn
or
$ npm install
```

#### 运行
```bash
$ yarn start
or
$ npm run start
```
所用到的工具自行学习了解：
1. [yarn](https://yarn.bootcss.com/)
2. [node](https://nodejs.org/zh-cn/)
3. [react](https://reactjs.org/)
